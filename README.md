# Nest Starter

## Description

Starter kit developed by - [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Deploy

```bash
# deploy app to the ZEIT Now
$ npm run deploy
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
